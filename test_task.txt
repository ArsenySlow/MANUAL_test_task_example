Тестовое задание

Для вас была подготовлена тестовая страница лендинга.

Составьте, пожалуйста, три списка правок: 
явные дефекты на ваш взгляд
спорные моменты для вас, которые нужно исправить
список рекомендаций по улучшению страницы 

Правила оформления:
Дефекты оформить в формате: шаги, ожидаемый результат, фактический результат. 
Рекомендации можете оформить в свободной форме, как считаете удобным.
